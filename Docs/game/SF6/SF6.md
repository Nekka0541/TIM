## アケコン修行
- つまみ持ちとワイン持ちの中間位
- レバーの根っこをもつといいかも（ななめへの誤入力が減らせそう）
  - 236KK キャンセル　236236Kが出し易い

    - 下入力は人差し指担当
    ~~- 肘を締める、手の縦に水平になるように~~

- 余り力入れない方がいい

- ヒット確認練習方法
